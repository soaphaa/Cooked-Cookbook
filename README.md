# Cooked-Cookbook 
---
By @alyn-istic-archives @clairezhu7 @soaphaa @nia359 
This is a recipe website built for our Grade 12 Computer Science project. Designed with a strong focus on user experience with numerous interactive features and a hand-drawn picnic-themed ui.

Unlike many recipe websites that are cluttered with ads and hard to follow layouts, Cooked Cookbook is designed to improve your cooking experience!
